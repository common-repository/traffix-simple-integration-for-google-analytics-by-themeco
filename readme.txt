== Description ==

Traffix - Simple Integration for Google Analytics by Themeco is a simple and easy to tool, drop in your Google Analytics code snippet, select where you’d like it to be output and you’re good to go! Google Analytics couldn’t get any easier. 

== Installation ==

1. Activate the plugin
2. Go to Settings > Traffix and check the enable box
3. Enter in your Google Analytics code
4. Pick where you want the code to output (header or footer)
5. Save. It’s as simple as that!
